# **App Name**: Elleyhill Power Connect

## Core Features:

- Company Profile: Detail the mission, vision, and values of Elleyhill Power Zambia.
- Service Catalog: Outline the range of power solutions and services offered.
- Project Portfolio: Showcase completed projects with descriptions and images.
- Inquiry Submission: Allow users to submit inquiries through a contact form.

## Style Guidelines:

- Primary color: Vibrant green (#50C878) to represent energy and sustainability.
- Background color: Light gray (#F0F0F0), almost white, for a clean and professional feel.
- Accent color: Silver (#C0C0C0) to add a touch of sophistication and modernity.
- Body and headline font: 'Inter' sans-serif for a clean and modern look, suitable for both headings and body text.
- Use professional, clear icons to represent different services and aspects of the company.
- Implement a grid-based layout for consistent content presentation across all pages.
- Subtle transition effects when navigating between sections.
